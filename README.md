# Blakes-Can-Recycling

Blakes-Can-Recycling is a robust and efficient platform to manage and promote can recycling activities. The repository includes tools, documentation, and practical insights to run recycling campaigns successfully.

## Features
- Track recyclable items and recycling statistics.
- Educational resources to encourage recycling.
- Tools to analyze and optimize recycling programs.

## Getting Started
### Prerequisites
Ensure you have the following installed on your system:
- Python 3.8+
- pip package manager

### Installation
1. Clone the repo:
   ```
   git clone https://github.com/blakemeredith79/Blakes-Can-recycling.git
   ```
2. Navigate to the project directory:
   ```
   cd Blakes-Can-recycling
   ```
3. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

### Usage
Run the application with:
```
python app.py
```
